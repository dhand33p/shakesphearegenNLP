The Project

Great Writing Style

Shakespeare AI is a custom trained GPT-2 model trained to write Shakespearean sonnets.

Interactive

Shakespeare AI will write a sonnet based off of any prompt you provide it.

Timeline

Shakespeare AI was completed by a teenager over the course of a week.

Extensions

Some extensions to this project include deploying it to the web, building a Discord Bot to share it with the world, or creating a bot that can mimic the writing style of other famous poets.
